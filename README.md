# My first RPG game 🧙
My first RPG Game in Python! 🕹️ Using Object Oriented Programming.  
![dragon](dragon.gif)

## Concepts applied 🤓:
- The 4 pilars of Object Oriented Programming 
  - Abstraction
  - Encapsulation
  - Inheritance
  - Polymorphism
- Setters and Getters
- Design Patterns

In the town you live people are dying of uknown disease. You still don't know why 🤨...
